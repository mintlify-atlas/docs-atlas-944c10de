# Quick start

This guide will walk you through creating your first ModernSlider. We'll build a brightness control slider similar to the one in macOS Control Center.

## Basic implementation

<Steps>
  <Step title="Import ModernSlider">
    First, import the ModernSlider library in your SwiftUI view file:
    
    ```swift
    import SwiftUI
    import ModernSlider
    ```
  </Step>
  
  <Step title="Create a state variable">
    Add a `@State` variable to hold the slider's value:
    
    ```swift
    @State private var brightness: Double = 20.0
    ```
  </Step>
  
  <Step title="Add the ModernSlider to your view">
    Use ModernSlider in your view's body with the minimum required parameters:
    
    ```swift
    var body: some View {
        ModernSlider(
            "Brightness",
            systemImage: "sun.max.fill",
            value: $brightness
        )
    }
    ```
  </Step>
</Steps>

## Complete example

Here's a complete working example showing a brightness slider with value change callbacks:

```swift ContentView.swift
import SwiftUI
import ModernSlider

struct ContentView: View {
    @State private var brightness: Double = 20.0

    var body: some View {
        VStack(spacing: 8) {
            ModernSlider(
                "Brightness",
                systemImage: "sun.max.fill",
                sliderWidth: 250,
                sliderHeight: 25,
                value: $brightness,
                onChange: { newValue in
                    print("Brightness onChange: \(newValue)")
                }
            )
            
            Text("Brightness: \(Int(brightness))")
                .font(.subheadline)
                .foregroundColor(.gray)
        }
    }
}
```

## Key parameters

ModernSlider offers several customization options:

### Required parameters

- `systemImage`: SF Symbol name for the slider thumb icon (e.g., `"sun.max.fill"`)
- `value`: A binding to a `Double` that stores the current slider value

### Optional parameters

- `title`: Optional label displayed above the slider (e.g., `"Brightness"`)
- `sliderWidth`: Width of the slider track (default: `250`)
- `sliderHeight`: Height of the slider track and thumb (default: `25`)
- `sliderColor`: Color of the slider fill and thumb (default: `.white`)
- `range`: Value range as `ClosedRange<Double>` (default: `0...100`)
- `onChange`: Closure called when the slider value changes
- `onChangeEnd`: Closure called when dragging ends

## Advanced examples

### Custom color and range

Create a temperature slider with a custom color and range:

```swift
@State private var temperature: Double = 60.0

ModernSlider(
    "Temperature",
    systemImage: "thermometer",
    sliderWidth: 300,
    sliderHeight: 35,
    sliderColor: .blue,
    value: $temperature,
    in: -40...120,
    onChangeEnd: { newValue in
        print("Temperature onChangeEnd: \(newValue)")
    }
)
```

### Volume control

Create a volume slider with green color:

```swift
@State private var volume: Double = 40.0

ModernSlider(
    "Volume",
    systemImage: "speaker.wave.1.fill",
    sliderWidth: 275,
    sliderHeight: 25,
    sliderColor: .green,
    value: $volume
)
```

### Slider without title

You can omit the title to create a more compact slider:

```swift
@State private var sliderValue: Double = 80.0

ModernSlider(
    systemImage: "circle",
    sliderWidth: 325,
    sliderHeight: 30,
    value: $sliderValue
)
```

<Note>
  When you omit the title, ModernSlider automatically removes the background material and shadow for a cleaner look.
</Note>

## Callbacks explained

### onChange callback

The `onChange` callback is triggered continuously as the user drags the slider:

```swift
ModernSlider(
    "Brightness",
    systemImage: "sun.max.fill",
    value: $brightness,
    onChange: { newValue in
        // Called continuously while dragging
        print("Current value: \(newValue)")
    }
)
```

### onChangeEnd callback

The `onChangeEnd` callback is triggered only when the user finishes dragging:

```swift
ModernSlider(
    "Brightness",
    systemImage: "sun.max.fill",
    value: $brightness,
    onChangeEnd: { finalValue in
        // Called once when dragging ends
        print("Final value: \(finalValue)")
    }
)
```

<Info>
  Use `onChange` for real-time updates (like adjusting brightness) and `onChangeEnd` for actions that should only happen once (like saving to UserDefaults).
</Info>
